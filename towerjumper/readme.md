#Tower Jumper - UNFINISHED

Ein kleines Jump'n Run, in dem es gilt, einen Turm möglichst schnell zu erklimmen, ohne herunterzufallen. Je schneller man sich bewegt, desto höher springt - oder fällt - man auch.

Das Spiel ist größtenteils ein Probelauf, wie sich ein recht komplexer JavaScript-Loop mit puren DOM-Elementen als Simulation schlägt.
